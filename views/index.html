Image abstraction api layer project

You can check recent searches with exact time . Just add "/api/latest/imagesearch/" at the end of url

To make searches using this api tool. Add this "/api/imagesearch/:id" at the end of url. For example , /api/imagesearch/superman?offset=1


Make sure, you give value to offset. It can be either 1,2 or 3. If you give any other value, it will return an error. In above example "superman" is keyword. You can replace it with any keyword you will like to use
